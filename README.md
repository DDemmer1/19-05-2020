# 19-05-2020 SoSe20 - 2D Array Wiederholung

Übungsaufgabe vom 12.05.2020 für die Studierenden des Java II Seminars der Universität zu Köln

### 📝 Aufgabe:

Ziel dieser Aufgabe ist es ein Programm zu schreiben, dass fertige Sudokus, in Form eines ```int[][] grid``` auf ihre Richtigkeit überprüft. Ladet euch dazu einfach das GitHub Repository in Eclipe um eine schon vorgefertigte Programmstruktur mit einem Sudoku ```int[][] grid``` zu erhalten, dass es zu überprüfen gilt.
